# PersonalAccountManagement

A simple Python-based banking system that tracks deposits, withdrawals, and transaction history.

## Features
- **Transaction Tracking**: Every deposit and withdrawal is logged with a timestamp.
- **Operator Overloading**: Use `+` and `-` symbols to perform transactions directly on the account object.
- **Account Management**: Update and retrieve account holder names and numbers.

## How to Run
1. Ensure you have Python 3.x installed.
2. Run the main script:
   ```bash
   python main.py
3. Run tests:
    ```bash
   python test_account.py
